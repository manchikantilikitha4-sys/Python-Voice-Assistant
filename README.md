# Python-Voice-Assistant
A Python-based Smart Voice Assistant that performs tasks such as telling time and date, opening websites, providing weather updates, managing reminders, AI chat, and system utilities using Python libraries and APIs.

 📌 Project Overview

This project is a Python-based Smart Voice Assistant developed as **Task 4** for the **Intern Alpha Python Internship**.

The assistant performs common desktop tasks, answers user queries, opens websites, manages reminders, provides weather updates, and supports AI-based conversations.

 🚀 Features

* 🕒 Tell current time
* 📅 Tell current date
* 📆 Tell current day
* 🌐 Open Google
* ▶️ Open YouTube
* 💻 Open GitHub
* 💼 Open LinkedIn
* 🗺️ Open Google Maps
* 🤖 Open ChatGPT
* 🌦️ Weather information
* 🧮 Calculator using WolframAlpha
* 💬 AI Chat using OpenAI
* 📝 Add reminders
* 📋 Show reminders
* 😂 Tell jokes
* 🔋 Battery status
* 📖 Help menu
* ❌ Exit assistant

 🛠️ Technologies Used

* Python 3
* pyttsx3
* requests
* wikipedia
* pywhatkit
* pyjokes
* psutil
* WolframAlpha API
* OpenAI API

 ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Python-Voice-Assistant.git
```

2. Install the required libraries.

```bash
pip install -r requirements.txt
```

3. Add your API keys in `assistant.py`.

* OpenAI API Key
* OpenWeather API Key
* WolframAlpha App ID

4. Run the program.

```bash
python assistant.py
```
